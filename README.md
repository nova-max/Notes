# AppNotas

A modern note-taking application with Google Drive sync and Gemini AI analysis.

## Prerequisites

- **Node.js**: You must have Node.js installed to run this project. Download it from [nodejs.org](https://nodejs.org/).

## Setup & Run

1.  Open a terminal in this folder (`AppNotas`).
2.  Install dependencies:
    ```bash
    npm install
    ```
3.  Start the development server:
    ```bash
    npm run dev
    ```
4.  Open your browser at the URL shown (usually `http://localhost:5173`).

## Features

- **Authentication**: Sign in with Google or use Guest Mode (Offline).
- **Dashboard**: Create, edit, and manage notes.
- **AI Analysis**: Click the sparkle icon to analyze notes with Gemini (Mocked for now).
- **Storage**: Local storage (Guest) or Google Drive (Cloud - *Coming Soon*).

## Tech Stack

- React + Vite
- Tailwind CSS
- Lucide Icons
