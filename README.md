# AppNotas

A modern note-taking application with Google Drive sync and Gemini AI analysis.

## Prerequisites

- **Node.js**: You must have Node.js installed to run this project. Download it from [nodejs.org](https://nodejs.org/).

## Setup & Run
- **Storage**: Local storage (Guest) or Google Drive (Cloud - *Coming Soon*).

## Tech Stack

- React + Vite
- Tailwind CSS
- Lucide Icons
